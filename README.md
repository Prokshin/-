# projectTeamSite
Небходимы node.js и npm.
Для запуска - в корне проекта выполнить 
```
npm install
```
Эта команда установит необходимые для работы сервера библиотеки. Затем выпольнить в корне:
```
node index.js
```
Эта команда запустит сервер.
